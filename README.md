# The Chai Station Launch

Welcome to the Chai Station project repository! 
This project is an artisanal chai café website to showcase handcrafted blends, bold flavors, and a great vibe for the Chai Station in Amravati.

## Getting Started

To get started with local development:

1. Install dependencies
```sh
npm install
```

2. Start the development server
```sh
npm run dev
```

## Features
- Interactive UI/UX
- Responsive design tailored for all devices
- Smooth animations and transitions
